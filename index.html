FaceFinder

FaceFinder is a desktop application that helps users search their own Google Drive photos using face recognition.

How it works

  • User signs in with their Google account.
  • The application downloads images directly from the user's Google Drive.
  • Face embeddings are generated locally on the user's computer.
  • The face index is stored locally.
  • No images are uploaded to any external server.
